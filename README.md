# 한국어교육 단어 검사기

서울대 한국어 교재를 기준으로, 입력한 한국어 문장이나 지문에 아직 배우지 않은 어휘와 표현이 섞여 있는지 확인하는 Windows용 데스크톱 프로그램입니다.

현재 **0.0.0**은 정식 출시 전 데모 버전입니다. 실제 사용 흐름과 검사 방식을 확인하기 위한 버전으로 봐 주세요.

## 이 프로그램으로 할 수 있는 일

- 한국어 문장이나 짧은 지문을 붙여넣고 어휘 수준을 검사할 수 있습니다.
- 목표 단원을 고르면, 그 단원까지 배운 어휘인지 확인할 수 있습니다.
- 아직 뒤 단원에 나오는 단어와 표현을 입력창에서 빨간색으로 확인할 수 있습니다.
- 교재 목록에서 확인되지 않은 표현은 `교재 외` 항목으로 따로 볼 수 있습니다.
- 검사 결과에서 단어가 처음 나오는 레벨과 단원을 확인할 수 있습니다.
- 자주 허용하고 싶은 표현은 허용 목록에 추가할 수 있습니다.
- 텍스트 파일, PDF, HWPX, HWP, DOCX 파일을 열어 검사할 수 있습니다.
- 어휘 검색 탭에서 특정 단어가 교재 어디에 나오는지 찾아볼 수 있습니다.

## 다운로드와 실행

[Releases](https://github.com/baggychani/Korean_Edu_Voca_Check/releases)에서 다음 중 하나를 받으면 됩니다.

| 파일 | 설명 |
| --- | --- |
| `KVocabGuard_v*_Setup.exe` | **설치형** — 권장. 시작 메뉴·바탕화면 바로가기 생성 |
| `KVocabGuard-windows-x64-portable-v*.zip` | **포터블** — 압축만 풀고 `KVocabGuard.exe` 실행 |

설치형을 쓰면 `KVocabGuard` 이름으로 시작 메뉴에 등록됩니다. (바로가기 이름에는 버전이 붙지 않습니다.)

Windows에서 처음 실행할 때 보안 안내가 나올 수 있습니다. 직접 내려받은 파일이 맞는지 확인한 뒤 실행해 주세요.

앱을 켜 두면 GitHub Releases에 새 버전이 올라왔을 때 업데이트 안내가 표시될 수 있습니다.

## 기본 사용법

1. 검사할 목표 단원을 선택합니다.
2. 텍스트 입력 칸에 한국어 문장이나 지문을 붙여넣습니다.
3. `텍스트 검사` 버튼을 누릅니다.
4. 아래 검사 결과에서 어휘와 표현을 확인합니다.

결과는 판정별로 묶여 표시됩니다. 같은 판정 안에서는 글에 나온 순서대로 정렬됩니다.

## 결과 판정의 뜻

| 판정 | 뜻 |
| --- | --- |
| 사용 가능 | 목표 단원까지 이미 나온 어휘나 표현입니다. |
| 아직 이릅니다 | 교재에는 있지만, 목표 단원보다 뒤에 나오는 어휘나 표현입니다. |
| 교재 외 · 난이도 높음 | 교재 목록에서 찾지 못했고, 학습자에게 어려울 가능성이 높은 표현입니다. |
| 교재 외 · 검토 필요 | 교재 목록에서 찾지 못해 사람이 확인해 보는 것이 좋은 표현입니다. |
| 교재 외 · 참고 | 교재 목록에는 없지만 큰 문제는 아닐 수 있는 표현입니다. |
| 허용 목록 | 사용자가 직접 허용한 표현입니다. |

## 예시

목표 단원을 `1A 5-1`로 두고 다음 문장을 검사한다고 가정해 보겠습니다.

```text
서울대학교 한국어 교재를 사고 싶은데 너무 가격이 비싸요.
```

검사기는 `가격`, `비싸다`처럼 뒤 단원에 나오는 단어를 찾아 `아직 이릅니다`로 표시할 수 있습니다. 반대로 목표 단원 안에서 이미 배운 표현은 `사용 가능`으로 처리됩니다.

## 파일 열기

`파일 열기` 버튼으로 문서 파일을 불러올 수 있습니다.

지원 형식:

- `.txt`
- `.pdf`
- `.hwpx`
- `.hwp`
- `.docx`

단, 스캔 이미지로만 되어 있는 PDF는 텍스트를 바로 읽지 못할 수 있습니다. 이 경우 OCR 처리가 필요합니다.

## 허용 목록

사람 이름, 학교 이름, 고유명사, 수업에서 일부러 쓰는 표현처럼 매번 경고할 필요가 없는 항목은 허용 목록에 추가할 수 있습니다. 허용 목록에 들어간 표현은 이후 검사에서 경고로 표시되지 않습니다.

교재 고정 등장인물 이름(테오, 마리, 나나 등)은 처음부터 허용 목록에 들어 있으며, 삭제할 수 없습니다. 사용자가 추가한 허용어는 프로그램을 껐다 켜도 유지됩니다.

## 어휘 검색

어휘 검색 탭에서는 단어를 직접 입력해 교재 안에서 처음 나오는 위치를 찾을 수 있습니다.

예를 들어 `전화`를 검색하면 `전화하다`, `전화번호`, `전화를 받다`처럼 관련 항목을 확인할 수 있습니다. 쉼표로 여러 표제어를 한 번에 검색할 수도 있습니다. (예: `라면, 가다`)

## 형태소 분석에 대하여

이 프로그램은 한국어 문장을 분석하기 위해 Kiwi 형태소 분석기를 사용합니다. 그래서 `먹고`, `보는`, `비싸요`처럼 문장 안에서 활용된 형태를 가능한 한 기본형으로 바꾸어 검사합니다.

다만 한국어 표현은 문맥에 따라 분석이 달라질 수 있습니다. 결과가 이상해 보이면 목표 단원, 원문 표현, 판정 내용을 함께 확인해 주세요.

## 현재 데모 버전에서 알아둘 점

- 교재 데이터는 계속 정리하고 검토하는 중입니다.
- 일부 표현은 실제 수업 맥락과 다르게 판정될 수 있습니다.
- 검사 결과는 최종 판단이 아니라 수업 자료를 다듬기 위한 참고 도구로 사용하는 것을 권장합니다.
- 정식 출시 전 데모 버전이므로 화면 구성과 판정 방식은 바뀔 수 있습니다.

## 권장 사용 상황

- 선생님이 수업 자료를 만들 때
- 학생에게 아직 어려운 단어가 들어갔는지 확인하고 싶을 때
- 교재 진도에 맞춰 예문이나 지문을 다듬고 싶을 때
- 특정 단어가 서울대 한국어 교재 어디쯤 나오는지 빠르게 찾고 싶을 때
